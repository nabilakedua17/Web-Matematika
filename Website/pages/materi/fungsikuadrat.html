FUNGSI KUADRAT

A. BENTUK UMUM

Fungsi kuadrat didefinisikan sebagai fungsi polinomial berderajat dua dengan bentuk:

f(x) = ax2 + bx + c

Syarat: a, b, c ∈ ℜ dan a ≠ 0.

B. PENYAJIAN & SIFAT GRAFIK

Grafik fungsi kuadrat adalah kurva mulus yang disebut parabola. Sifatnya ditentukan oleh:

  • Nilai a: Menentukan arah terbuka ( a > 0 ke atas, a < 0 ke bawah).
  • Diskriminan (D = b2 - 4ac): Menentukan titik potong dengan sumbu X.

C. SUMBU SIMETRI & NILAI OPTIMUM

Titik puncak atau titik balik parabola dinyatakan dengan koordinat (xp, yp).

Rumus Utama:
Sumbu Simetri:
xp =
-b 2a
Nilai Optimum:
yp =
-D 4a
Contoh Soal: f(x) = x2 - 4x - 5
A. Mencari Sumbu Simetri:
Diketahui a = 1, b = -4, c = -5.
xp = -(-4) / (2 × 1) = 4 / 2 = 2

B. Mencari Nilai Optimum:
Hitung D = b2 - 4ac
D = (-4)2 - 4(1)(-5) = 16 + 20 = 36
yp = -36 / (4 × 1) = -9

Maka, koordinat titik baliknya adalah (2, -9).

D. MENYUSUN RUMUS FUNGSI KUADRAT

Jika diketahui titik puncak (xp, yp) dan satu titik lain (x, y):

f(x) = a(x - xp)2 + yp
Contoh: Puncak (2, -3) dan melalui (0, 5)
1. Substitusi puncak: y = a(x - 2)2 - 3
2. Substitusi titik (0, 5) untuk mencari a:
   5 = a(0 - 2)2 - 3
   5 = 4a - 3 ⇒ 8 = 4a ⇒ a = 2
3. Rumus akhir: f(x) = 2(x - 2)2 - 3 = 2x2 - 8x + 5

LATIHAN MANDIRI

1. Tentukan sumbu simetri dari fungsi f(x) = 2x2 + 8x + 5.

2. Diketahui titik puncak (1, 2) melalui titik (0, 3), tentukan rumusnya.
