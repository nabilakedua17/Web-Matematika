BAB VI: PERSAMAAN KUADRAT

Persamaan kuadrat adalah suatu persamaan polinomial berorde dua. Bentuk visual dari persamaan ini dalam koordinat Kartesius akan membentuk kurva simetris yang disebut Parabola.

A. BENTUK UMUM PERSAMAAN KUADRAT

ax2 + bx + c = 0

Keterangan variabel dan konstanta:

  • x : Variabel (nilai yang ingin diketahui).
  • a, b : Koefisien bilangan real, dengan syarat a ≠ 0.
  • c : Konstanta atau suku tetap.

B. AKAR-AKAR PERSAMAAN KUADRAT

Akar-akar adalah nilai x yang memenuhi persamaan. Terdapat tiga metode utama penyelesaian:

1. Metode Pemfaktoran

Contoh Soal 1: Tentukan akar-akar dari persamaan x2 + 7x + 12 = 0.
1. Cari dua bilangan p dan q yang jika dikali = 12 dan dijumlah = 7.
2. Bilangan tersebut adalah 3 dan 4.
3. Bentuk faktor: (x + 3)(x + 4) = 0
4. Maka: x + 3 = 0 atau x + 4 = 0
Hasil: x1 = -3 atau x2 = -4

2. Rumus Kuadrat (Rumus ABC)

Digunakan jika persamaan sulit difaktorkan secara langsung.

x1,2 =
-b ± √(b2 - 4ac) 2a
Contoh Soal 2: Selesaikan 2x2 + 5x - 3 = 0 menggunakan Rumus ABC.
Diketahui: a = 2, b = 5, c = -3
x =
-5 ± √(52 - 4 · 2 · -3) 2 · 2
x =
-5 ± √(25 + 24) 4
=
-5 ± 7 4
Hasil: x1 = 0,5 atau x2 = -3

C. PERMASALAHAN AKAR-AKAR

1. Diskriminan (D)

Nilai Diskriminan menentukan jenis akar yang dihasilkan:

D = b2 - 4ac
  • D > 0 : Memiliki dua akar real yang berbeda.
  • D = 0 : Memiliki satu akar real (akar kembar).
  • D < 0 : Tidak memiliki akar real (akar imajiner).

2. Menyusun Persamaan Kuadrat Baru

Jika diketahui akar-akar x1 dan x2, maka persamaannya adalah:

x2 - (x1 + x2)x + (x1 · x2) = 0

D. CARA MEMBUAT GRAFIK FUNGSI KUADRAT

Untuk menggambar grafik parabola dari persamaan y = ax2 + bx + c, ikuti langkah-langkah sistematis berikut:

  1. Identifikasi Arah Terbuka Kurva:
    • Jika a > 0, kurva terbuka ke atas (seperti senyuman).
    • Jika a < 0, kurva terbuka ke bawah.
  2. Tentukan Titik Potong Sumbu X: Cari akar-akar persamaan saat y = 0. Titik koordinatnya: (x1, 0) dan (x2, 0).
  3. Tentukan Titik Potong Sumbu Y: Nilai x = 0, maka y = c. Titiknya: (0, c).
  4. Tentukan Sumbu Simetri: Gunakan rumus xp = -b / 2a.
  5. Tentukan Titik Puncak (Titik Ekstrem): Masukkan nilai xp ke dalam persamaan awal untuk mendapatkan yp. Koordinat puncak: (xp, yp).
  6. Hubungkan Titik: Gambar titik-titik tersebut pada koordinat Kartesius dan hubungkan dengan garis lengkung mulus.

LATIHAN MANDIRI

  1. Tentukan nilai Diskriminan dari persamaan x2 - 6x + 9 = 0!
  2. Tentukan titik puncak dari fungsi f(x) = x2 - 4x + 3!
  3. Susunlah persamaan kuadrat yang akar-akarnya adalah 2 dan 5!
