BAB V: SISTEM PERTIDAKSAMAAN LINEAR DUA VARIABEL (SPtLDV)

A. PENGERTIAN SPtLDV

Sistem Pertidaksamaan Linear Dua Variabel (SPtLDV) adalah kumpulan dari dua atau lebih pertidaksamaan linear yang memiliki dua variabel (biasanya x dan y). Semua pertidaksamaan dalam sistem ini harus dipenuhi secara bersamaan untuk menemukan solusi tunggal berupa daerah grafik.

Bentuk umum pertidaksamaan dalam sistem ini menggunakan tanda:

ax + by < c | ax + by > c | ax + by ≤ c | ax + by ≥ c

B. CIRI-CIRI UTAMA

  • Terdiri dari dua variabel yang berbeda (misalnya x dan y).
  • Pangkat tertinggi variabel adalah satu.
  • Himpunan penyelesaiannya berupa Daerah Himpunan Penyelesaian (DHP).

C. LANGKAH MENGGAMBAR GRAFIK SPtLDV

  1. Ubah ke Persamaan: Ganti tanda pertidaksamaan menjadi tanda sama dengan (=).
  2. Cari Titik Potong:
    • Potong sumbu X jika y = 0.
    • Potong sumbu Y jika x = 0.
  3. Tentukan Jenis Garis:
    • Gunakan Garis Penuh jika tanda memiliki sama dengan (≤ atau ≥).
    • Gunakan Garis Putus-putus jika tanda murni (< atau >).
  4. Lakukan Titik Uji (Arsiran): Ambil titik (0,0). Jika hasilnya benar, arsir ke arah (0,0). Jika salah, arsir ke arah sebaliknya.

D. PENYELESAIAN CONTOH KASUS TERPERINCI

Contoh 1: Masalah Toko Roti Roti cokelat (x) butuh 3 ons tepung, roti keju (y) butuh 2 ons tepung. Tepung tersedia 60 ons.
1. Identifikasi Variabel & Model: Misal x = Roti Cokelat, y = Roti Keju.
Model: 3x + 2y ≤ 60, dengan syarat x ≥ 0, y ≥ 0. 2. Mencari Titik Potong (Garis 3x + 2y = 60): • Jika x = 0 → 2y = 60 → y = 30. Titik potong (0, 30).
• Jika y = 0 → 3x = 60 → x = 20. Titik potong (20, 0). 3. Menentukan Arsiran: Uji titik (0,0): 3(0) + 2(0) ≤ 60 → 0 ≤ 60 (Benar).
Maka, daerah arsiran mengarah ke titik pusat (0,0).
Contoh 2: Pembelian Alat Tulis Pensil (x) Rp2.000, buku (y) Rp5.000. Uang maksimal Rp40.000.
1. Model Matematika: 2000x + 5000y ≤ 40000. Disederhanakan menjadi 2x + 5y ≤ 40.
Syarat: x ≥ 0, y ≥ 0. 2. Mencari Titik Potong (Garis 2x + 5y = 40): • Jika x = 0 → 5y = 40 → y = 8. Titik potong (0, 8).
• Jika y = 0 → 2x = 40 → x = 20. Titik potong (20, 0). 3. Menentukan Arsiran: Uji titik (0,0): 2(0) + 5(0) ≤ 40 → 0 ≤ 40 (Benar).
Contoh 3: Lahan Pertanian Lahan 100 m². Cabai (x) butuh 2 m² dan Tomat (y) butuh 4 m².
1. Model Matematika: 2x + 4y ≤ 100. Disederhanakan menjadi x + 2y ≤ 50.
Syarat: x ≥ 0, y ≥ 0. 2. Mencari Titik Potong (Garis x + 2y = 50): • Jika x = 0 → 2y = 50 → y = 25. Titik potong (0, 25).
• Jika y = 0 → x = 50. Titik potong (50, 0). 3. Jenis Garis: Karena menggunakan tanda ≤, maka grafik digambar dengan Garis Tegas (Penuh).

LATIHAN MANDIRI

  1. Soal Cerita: Harga nasi kuning Rp8.000 dan es teh Rp3.000. Uang tersedia Rp35.000. Tentukan model matematikanya dan cari titik potong sumbu X dan Y!
  2. Grafik: Gambarkan daerah penyelesaian dari sistem pertidaksamaan: x + y ≤ 6, x - y ≥ 2, x ≥ 0, y ≥ 0.
